# Desenvolvimento de API com FastAPI 🚀

Repositório destinado à documentação e código do projeto de desenvolvimento de uma API REST utilizando Python e o framework FastAPI.

## 📋 Sobre o Projeto
O objetivo principal foi a configuração de um ambiente de desenvolvimento isolado (`venv`) e a implementação de rotas estruturadas para retorno de dados em formato JSON, seguindo boas práticas de desenvolvimento back-end.

## 🛠️ Tecnologias Utilizadas
- **Python 3.12**
- **FastAPI**: Framework para construção de APIs.
- **Uvicorn**: Servidor ASGI para execução.
- **Git/GitHub**: Versionamento e documentação.
